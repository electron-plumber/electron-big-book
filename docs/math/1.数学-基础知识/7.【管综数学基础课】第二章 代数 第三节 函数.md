###### 〖例4〗函数𝑦=𝑥²−2𝑥−5在区间−1,5上的最大值与最小值之差为( )
完全平方公式:  
(a+b)²=a²+b²+2ab  
(a-b)²=a²-2ab+b²  
𝑦=𝑥²−2𝑥  
配一元二次函数:  
=> 2b=-2  
=> b=-1  
𝑦=𝑥²−2𝑥+1-6 = (𝑥−1)²−6 ≥ (0-6 = -6)  
对称轴: 𝑥=1  
(𝑥-1)²这个平方项的系数隐含地为1(即a=1):开口向上  

min: 当𝑥=1时  𝑦=-6  
max: 当𝑥=5时  𝑦=(5-1)²−6 = 10  
=> 10-(-6)  
=> 16  

或者根据函数性质:  
𝑦=a𝑥²+b𝑥+c  
𝑥=-b/2a=-(-2/2x1)=1  
a>0:开口向上  
对称轴: 当𝑥=1时  𝑦 = 1²-2x1-5 = 1-2-5 = -6  
当𝑥=-1时  𝑦 = (-1)²-2x(-1)-5 = 1+2-5 = -2  
当𝑥=5时  𝑦= 5²-2x5-5 = 25-10-5 = 10  

min: 𝑦=-6  
max: 𝑦=10  
=> 10-(-6)  
=> 16
